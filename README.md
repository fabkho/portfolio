# Fabian Kirchhoff

Personal portfolio and blog at [fabkho.dev](https://fabkho.dev).

Built with Nuxt, Nuxt Content, Tailwind CSS, and NuxtHub.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Runs at `http://localhost:3000`.

## Build

```bash
pnpm build
```

## Preview

```bash
pnpm preview
```

## Content sync

```bash
pnpm sync
```
